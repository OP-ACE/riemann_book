Ways to view the notebooks:

  • View the notebooks rendered as html (Note that interactive widgets do not work when viewed this way.)

  • View the GitHub repository of notebooks
    Scroll to the bottom where the README is displayed for more information about this book and how to run the notebooks.
    Note that the FA16 branch of the repository contains the notebooks corresponding to the printed book, while the master branch may have been updated. (See also the Errata).

  • Start up a notebook server on binder running these notebooks. (This may take several minutes to launch.)
