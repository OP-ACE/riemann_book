Errata

  • Some of the links to the GitHub repository (in footnotes in the printed version of the book) refer to the master branch. In the future master may be updated.
    The stable way to reach exactly the version that goes with the published book is to instead use the FA16 branch of the repository, i.e. replace master with FA16 in any footnotes that refer to the repository.

    For example, instead of

    • https://github.com/clawpack/riemann_book/blob/master/exact_solvers/shallow_water.py
    use
    • https://github.com/clawpack/riemann_book/blob/FA16/exact_solvers/shallow_water.py

    The FA16 branch of the repository can be browsed starting from https://github.com/clawpack/riemann_book/tree/FA16/.

  • To launch binder based on the FA16 branch rather than the master branch, use this link:
    • https://mybinder.org/v2/gh/clawpack/riemann_book/FA16
    or better yet this one, which opens the Index.ipynb notebook:
    • https://mybinder.org/v2/gh/clawpack/riemann_book/FA16?filepath=riemann_book%2FIndex.ipynb
